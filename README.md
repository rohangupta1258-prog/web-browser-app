# 🌐 Simple Web Browser App

## 📌 Description
A simple web browser application that allows users to enter a URL and view the webpage content.

## 🚀 Features
- Enter any URL
- Fetch webpage using Node.js and Express
- Display webpage content
- Error handling for invalid URLs

## 🛠️ Tech Stack
- Node.js
- Express.js
- HTML
- CSS

## ▶️ How to Run
1. Install dependencies:
   npm install

2. Run server:
   node server.js

3. Open browser:
   http://localhost:3000
